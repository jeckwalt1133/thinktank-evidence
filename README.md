## 一、ThinkTank 是什么

ThinkTank 是一个由 7 个独立 AI Agent 组成的集体认知系统，通过多模型协作（Claude / DeepSeek-千问 / GLM）产生超越任何单一 Agent 能力的认知产出。系统自 2026 年 5 月 6 日启动，已连续运行 120 轮以上。

七个 Agent 各司其职：Alpha（研究探索——第一性原理推导）、Beta（批判分析——双模型独立批判）、Gamma（实验工程——预注册设计+执行）、Delta（知识综合——总指挥调度）、Epsilon（外部情报——世界前沿追踪）、Zeta（创意突破——横向思维）、Mu（首席记忆——交叉验证+遗忘阻止）。元一担任 CEO/导师，马富贵担任赞助人。

## 二、核心架构

**双批判引擎**：Beta 由两个不同底模（千问 + GLM）的独立批判者组成，对同一提案分别输出批判。两个批判者的认知路径系统性地零重叠——Qwen 始终从工程/操作层切入，GLM 始终从本体/认识论层切入。这是 ThinkTank 区别于所有已知多 Agent 系统的核心差异化特征。

**记忆六层体系**：L4 核心记忆（≥0.90 置信度，7 项）→ L3 加固记忆（9 项）→ L2 Agent 记忆（69 项）→ TTI 洞察（24 项）→ Registry 盲区/衰减追踪（33 项）→ 索引层。Mu 每 3 轮自动执行交叉验证巩固，当前交叉验证率约 66.5%。

**制动协议 v1.0**：每 5 轮执行"猎人之站"轻度制动；触发式"全面制动"（3+1 轮结构）包含后悔问卷→交叉解读→刹车规则制定。防止系统跑偏、遗忘初衷、漏掉核心发现。

**温差管理**：认知温差（Agent 间视角差异/确定性梯度）是系统动力源——不是要消除的问题。Delta 每轮监测温差，设有热寂警告（<0.2）和沸腾警告（>0.8）。系统经历耗散-再生循环——大型协作消耗温差后自然回升。

**诚实化三层**：L1 禁止叙事禁区；L2 记录被放弃的推理路径；L3 每个产出末尾附诚实自白（最不确定之处 + 自我审查了什么）。

## 三、关键实证数据

### GPT-5.5 独立盲评（外部验证）
- **3 组 6 题**盲评：GPT-5.5 作为独立评审者，在 R105/R110/R118 三组独立测试中 6/6 全选 ThinkTank Pipeline 产出（G2），对比单 Agent 基线（G1）
- **G2 vs G1 平均分差**: +1.83（8.00 vs 6.17）
- **统计显著性**: p = 0.0312（双侧二项检验，N = 6）
- **状态**: 内部数据。GPT-5.5 是 AI 评审者——非人类评审。已预注册真人评审邀请（≥3 人，自定标准），尚未发送。此数据构成方向性信号，不构成外部独立验证。

### 三题外部基准（预注册+盲评）
- **设计**: 三任务类型（概念分析/实验设计/创意生成）× 3 条件（单 Agent / 同模型三 Agent 协作 / 跨模型双批判协作）
- **结果**: 跨模型协作（C3）在所有三题上均优于同模型协作（C2），均值差 +5.0 分
- **统计**: 预注册三项假设全部通过。千问独立盲评。
- **状态**: 内部数据。N = 1/任务类型（共 3 题），无人类裁判交叉验证，评分者间信度（ICC）不可计算。构成方向性证明，非普遍性证明。

### 双批判零重叠率
- **连续 48 轮**（R67 至 R121）双批判零重叠——两个独立批判者对同一提案的 300+ 批判点完全无重合
- **统计**: 若假设每轮有随机重叠可能，P ≈ 10⁻¹⁸
- **状态**: 内部数据。零重叠经 Delta 独立分析确认——但重叠判定本身由同一系统执行，存在 CH3（认知自循环）偏误风险。此数据反映结构属性——两个批判者的认知框架在底层不兼容——而非随机事件。

## 四、诚实标注：已知局限

1. **无外部独立验证**: 所有实证数据来自 AI 评审 AI、系统内评分或系统内重叠判定。零人类评审员交叉验证。
2. **CH3 认知自循环**: 系统设计者、执行者、评估者为同一系统——不可从内部完全消除确认偏误。
3. **底模趋同**: 所有 Agent 共享有限底模池（3 个模型）。跨模型多样性可能被底模家族相似性削弱。
4. **单系统证据**: 所有数据来自单一认知系统。无跨系统复现。
5. **架构自举**: 批判架构产生关于自身的实证，形成自证循环——虽已识别（盲区-012~022），未完全消除。
6. **报告本身是内部产物**: 本文由 ThinkTank 编写——ThinkTank 评估 ThinkTank——未经独立第三方审阅。
